     - Influence
      - Alpha : 1.0

Internal Renderer settings

     - World
      - Real Sky : Checked
     - Environment Lighting : Checked
      - Environment Color : Sky Texture
     - Gather
      - Samples : ~ 12 ~ 16 +

     - World Texture
      - Mapping : AngMap / Equirectangular ?
      - Influnce : Horizon = 1.0

HDRI / Env Sphere Painting

      create sphere
      unwrap with sphere projection
      filp normals
      change view to texture
      material shadeless
      texture paint mode
      add paint slot 2048x2
      Properties > Shading > Backface Culling : On
      In UV / Image Editor : Switch to Paint Mode

      Save when done.
