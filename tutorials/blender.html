HDRI / Env Sphere Painting

      create sphere
      unwrap with sphere projection
      filp normals
      change view to texture
      material shadeless
      texture paint mode
      add paint slot 2048x2
      Properties > Shading > Backface Culling : On
      In UV / Image Editor : Switch to Paint Mode

      Save when done.

Rigging

      make rig
      select ctrl bones and clear parent
      make pole targets
      name all bones (With .L)
      select ctrl bones and remove Deform(Under Bone)

      Parent Foot to Foot Ctrl

      Setup IK
       Select Ctrl, Shift + Select last bone in chain
       Ctrl + Shift + C to add IK
       Set chain length & pole target ( Armature -> BoneName )
       Adjust Pole Angle as necessary

      Duplicate and flip - Names Too 
      cursor at center
      shift + d to copy
      s x -1
      Armature > Flip Names

      Skin
      Select Mesh + Shift select armature
      Ctrl + P > With Automatic Weights

      Parent other bones as appropriate
